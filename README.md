# CHA◎ Game Website

A visually stunning website for the CHA◎ (Legends of the Asian Gods: Battle for Heaven) board game, featuring ancient Chinese mythology, I Ching divination, and strategic social deduction gameplay.

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Interactive Character Gallery**: Filter and explore 14 unique characters with detailed information
- **Game Information**: Complete overview of gameplay, rules, and philosophy
- **Visual Appeal**: Uses the provided `dajissssss-001.jpg` as a background with elegant overlays
- **Modern UI**: Smooth animations, interactive elements, and intuitive navigation

## How to Use

1. Open `index.html` in any modern web browser
2. Navigate using the menu or scroll through sections
3. Click character cards to view detailed information
4. Use filter buttons to view specific factions
5. The site is fully responsive - try it on different screen sizes

## Game Concept

CHA◎ is a 6-10 player social deduction game set in the mythological world of the *Investiture of the Gods*. Players take roles of gods, immortals, demons, and mortals with unique abilities tied to I Ching hexagrams. The game features:
- Night actions and day discussions/voting
- Three factions with different victory conditions
- I Ching divination integration
- "Ascension Scroll" mechanic for eliminated players
